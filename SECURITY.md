# Security Policy

## Reporting a Vulnerability

We take the security of SeenThis AI Hub seriously. If you find a security vulnerability, please report it to us via email:

**seenthiskr@gmail.com**

Please do not report security vulnerabilities via public GitHub issues.

## Scope
This policy applies to the SeenThis AI Hub API, MCP server, and related documentation. 
Any findings related to the main `seenthis.kr` platform should also be reported to the same address.
